# IMECursor 🎨

> **Windows IME 입력 상태를 감지하여 마우스 커서 색상과 트레이 아이콘을 실시간으로 변경해 주는 유틸리티입니다.**

영문 대/소문자, 한글은 물론 **팔리어(Pāḷi) 자판 상태**까지 완벽하게 감지하여, 타이핑하기 전에 현재 어떤 자판이 활성화되어 있는지 마우스 포인터와 작업 표시줄을 통해 직관적으로 확인할 수 있습니다. 특히 엑셀이나 아래아한글 환경에서 작업 효율을 극대화합니다.

---

## ✨ 주요 기능 (Key Features)

* **IME 상태별 마우스 커서 색상 변경**
  * 일반 화살표, I-Beam(텍스트 입력), Cross(십자) 커서에 모두 대응하여 실시간 색상 반영
* **팔리어(Pāḷi) 입력 상태 완벽 지원**
  * 팔리어 소문자(`p`) 상태일 때, 트레이 아이콘에 선명하고 아름다운 윈도우 내장 필기체(`Segoe Script` 기반 Multi-Overdraw Extra Bold)를 적용하여 시인성 확보
* **특정 오피스 프로그램 전용 표시기(Indicator)**
  * Microsoft Excel 및 아래아한글(HWP) 실행 시, 마우스 포인터 우하단에 상태별 색상을 가진 작은 원(Mini Indicator)을 띄워 현재 자판을 상시 확인 가능 (트레이 메뉴에서 온/오프 가능)
* **안정적인 시스템 통합**
  * 중복 실행 방지(Mutex) 처리 및 프로그램 종료 시 윈도우 기본 커서 상태 자동 복원

## 🎨 자판 상태별 색상 매핑

| 입력 상태 | 트레이 아이콘 표시 | 커서 / 표시기 색상 |
| :--- | :---: | :---: |
| **영어 소문자** | `e` | ⚪ 화이트 (기본) |
| **영어 대문자** | `E` | 🔷 딥 스카이 블루 |
| **한국어 (한글)** | `K` | 🔴 레드 |
| **팔리어 소문자** | *`p` (필기체)* | 🔸 오렌지 (`#FFAF00`) |
| **팔리어 대문자** | `P` | 🟢 라임 |

## 🚀 시작하기 (How to Use)

### 실행 파일 다운로드
오른쪽의 **[Releases]** 탭에서 최신 버전의 `.zip` 파일을 다운로드한 뒤, 압축을 풀고 `IMECursor.exe`를 실행하시면 즉시 시스템 트레이에 상주하며 작동합니다.

### 개발 환경 및 빌드 (Requirements)
* **OS:** Windows 10 / 11
* **Framework:** .NET 8.0-windows
* **Language:** C# 12
* **IDE:** Visual Studio 2022

```bash
# 저장소 클론
git clone [https://github.com/stonkim93/IMECursor.git](https://github.com/유저네임/IMECursor.git)

# 빌드 및 실행
dotnet build
dotnet run
